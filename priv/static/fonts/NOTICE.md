**SIL Open Font License, Version 1.1** (`OFL-1.1`). They are self-hosted (served from `/fonts`,

`font-src 'self'`.

|---|---|---|---|
| `hanken-grotesk-latin.woff2` | Hanken Grotesk | © Alfredo Marco Pradil | https://fonts.google.com/specimen/Hanken+Grotesk |
| `baloo2-latin.woff2` | Baloo 2 | © Ek Type | https://fonts.google.com/specimen/Baloo+2 |
| `jetbrains-mono-latin.woff2` | JetBrains Mono | © 2020 The JetBrains Mono Project Authors | https://fonts.google.com/specimen/JetBrains+Mono |

The Hanken Grotesk and JetBrains Mono files are variable fonts covering the full weight axis;
the Baloo 2 file is the bold display weight used for the wordmark. Each has been subset to the
Google Fonts `latin` unicode range to keep the bytes small.

The full OFL-1.1 text accompanies these files in [`OFL.txt`](OFL.txt) (also available with a FAQ at
https://openfontlicense.org), satisfying the license's requirement that it travel with the fonts.
